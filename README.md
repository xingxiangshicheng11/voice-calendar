# Voice Calendar Pro - 智能语音日历助手

## 📋 项目概述

一款基于 Web 技术的智能语音日历助手，支持通过语音或手动方式管理日程事件，提供纪念日主题换肤与日期特效功能。

- **开发周期**：2.5 天（60 小时）
- **技术栈**：HTML5 + CSS3 + 原生 JavaScript + Web Speech API + localStorage
- **核心功能**：语音增删查事件 + 纪念日主题换肤 + 日期特效

## ✨ 功能清单

| 模块 | 功能点 | 优先级 |
|------|--------|--------|
| 日历基础 | 网格渲染、月份切换、今日高亮 | P0 |
| 事件管理 | 手动/语音添加、删除、查询事件 | P0 |
| 数据存储 | localStorage 持久化 | P0 |
| 纪念日系统 | 预设/添加纪念日、格子换肤 | P1 |
| 主题特效 | 当天全局主题、庆祝动画 | P1 |
| 语音扩展 | 语音添加纪念日 | P2 |

## 🚀 快速开始

### 1. 启动本地服务器

语音识别功能依赖本地 HTTP 服务（不能直接用浏览器打开 `index.html` 文件），任选一种方式：

```bash
# 方式一：Python（推荐）
python -m http.server 8080

# 方式二：VS Code Live Server 插件
# 安装 Live Server 后，右键 index.html → Open with Live Server
```

### 2. 打开浏览器

在浏览器地址栏访问 `http://localhost:8080`（或 Live Server 提示的地址）。

### 3. 浏览器兼容性

| 浏览器 | 语音识别 | 说明 |
|--------|---------|------|
| Microsoft Edge | ✅ 正常 | **推荐**，语音识别走 Azure 服务，在中国大陆可用 |
| Google Chrome | ⚠️ 需 VPN | 语音识别依赖 Google 服务，在中国大陆需要 VPN |
| Firefox / Safari | ❌ 不支持 | Web Speech API 支持有限 |

> **提示**：如果语音识别失败并报错 `network`，说明浏览器无法连接语音服务器，请切换到 Edge 浏览器。

## 🎯 功能详情

### P0 - 核心功能

- **日历网格渲染**：按月生成日期网格，支持月份前后切换
- **今日高亮**：自动标记当天日期
- **事件管理**：点击日期添加事件，支持事件删除与语音查询
- **数据持久化**：所有事件数据保存在 localStorage，刷新不丢失

### P1 - 纪念日与主题

- **纪念日系统**：预设常见纪念日，支持用户自定义添加
- **格子换肤**：纪念日对应日期格子更换主题皮肤
- **全局主题**：当天匹配纪念日时应用全局主题色
- **庆祝动画**：纪念日触发庆祝动效

### P2 - 语音扩展

- **语音添加纪念日**：通过语音指令添加纪念日
