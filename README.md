# 智能语音日历

基于 Web 技术的语音交互日历，支持语音和手动方式管理事件与纪念日。

**技术栈：** HTML5 + CSS3 + 原生 JavaScript + Web Speech API + localStorage

---

## 功能

### 🎤 语音命令

| 分类 | 示例 |
|------|------|
| 添加事件 | `明天下午3点开会`、`下周一交报告`、`这周五去吃火锅` |
| 添加纪念日 | `记住5月20日结婚纪念日`、`今天得奖了，纪念一下` |
| 修改事件 | `把明天的会议改到4月3号` |
| 查询清单 | `今天有什么安排`、`这个月都有什么事`、`下周三有什么事` |
| 删除事件 | `删除明天的事件` |

支持多种日期格式：今天/明天/后天、上周/这周/下周、农历、中文数字等。

### 📅 日历

- 月历网格，月份切换
- 今日高亮
- 公历节日自动标注
- 农历日期显示
- 事件红点标记
- 纪念日图标与名称
- 粉色庆祝特效（匹配纪念日/节日时触发）
- 点击格子弹出当日完整清单

### 📋 事件管理

- 语音添加 / 手动添加（右侧"＋添加事件"按钮）
- 编辑弹窗：修改名称、日期、时间、备注
- 删除
- 语音原话自动存入备注

### 🎂 纪念日管理

- 语音添加 / 手动添加（右侧表单，支持图标选择）
- 编辑弹窗：修改名称、日期、图标、备注
- 删除
- 智能图标匹配（生日→🎂、结婚→💍、得奖→🏆 等）

### 🔍 摘要弹窗

点击日历格子弹出，显示当天全部：
- 事件（含时间、备注、✏️编辑）
- 节日
- 纪念日（含备注、✏️编辑）
- 支持语音播报清单

---

## 使用

直接在浏览器打开 `index.html`，推荐 Chrome。

点击麦克风开始录音，说话后自动识别执行。也可点击右侧按钮手动添加。

所有数据保存在浏览器 localStorage，刷新不丢失。
